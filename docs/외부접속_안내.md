# 인터넷 어디서나 접속 (2번 방법)

와이파이가 달라도(LTE, 다른 집·회사) **https 주소**만 알면 누구나 사이트를 볼 수 있습니다.

## 한 번에 시작 (가장 쉬움)

1. 사이트 실행 (아직 안 했다면):
   ```powershell
   cd c:\Users\psw00\OneDrive\Desktop\docker_exam2
   docker compose up -d
   ```

2. **공개 주소 만들기** — 아래 중 하나만 실행:
   - 탐색기에서 **`공개주소_시작.bat`** 더블클릭
   - 또는 PowerShell:
     ```powershell
     .\scripts\start-public-url-host.ps1
     ```
   - 주소가 만들어지면 **Google Chrome**으로 자동 열립니다.
   - 브라우저 없이 주소만: `.\scripts\start-public-url-host.ps1 -NoBrowser`
   - 이미 만든 주소를 크롬으로: **`크롬으로_열기.bat`**

3. 화면에 나오는 주소(또는 **`PUBLIC_URL.txt`** 파일)를 휴대폰·친구에게 공유  
   예: `https://xxxx.trycloudflare.com`

4. **끌 때**:
   ```powershell
   .\scripts\stop-public-url.ps1
   ```

## 꼭 알아둘 것

| 항목 | 설명 |
|------|------|
| PC 전원 | PC가 켜져 있고 `docker compose up -d` + 터널이 실행 중이어야 접속됩니다 |
| 주소 변경 | 터널을 다시 켤 때마다 주소가 바뀔 수 있습니다 |
| 용도 | 시연·확인용 (상시 서비스는 클라우드 배포 권장) |

## cloudflared가 없을 때

```powershell
winget install Cloudflare.cloudflared
```

설치 후 `공개주소_시작.bat` 을 다시 실행하세요.
