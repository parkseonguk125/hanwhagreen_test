# reference — 원본 사이트 참고 자료

`original-hanwhagreen/` 안 파일은 [hanwhagreen.com](https://hanwhagreen.com/) 레이아웃·CSS를 맞출 때 잠깐 저장해 둔 **참고용**입니다.

- 앱 실행에는 **필요 없습니다**.
- 실제 화면·스타일은 `frontend/src/` 에 있습니다.
- 삭제해도 Docker 실행에는 영향 없습니다.

| 파일 패턴 | 내용 |
|-----------|------|
| `tmp_*.html` | 원본 HTML 조각 |
| `tmp_*.css` | 원본 CSS 조각 |
