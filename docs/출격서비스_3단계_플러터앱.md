# 출격 서비스 3단계 — Flutter 앱 (초보자용)

> **완료 내용**: GPS·사진·출격 작성 폼 → `/api/attendance` 전송

---

## 앱 기능

| 기능 | 설명 |
|------|------|
| 작업일·작성자·인원·작업내용 | 폼 입력 |
| **현재 위치 저장** | GPS 좌표 + 주소(가능 시) |
| **네이버 지도에서 보기** | 외부 네이버 지도 앱/브라우저 (SDK 키 없이도 가능) |
| **카메라 / 갤러리** | 현장 사진 첨부 |
| **출격 등록** | 홈페이지 출결서비스에 즉시 반영 |

네이버 **Mobile SDK Client ID**가 없어도 GPS + 외부 지도 링크로 동작합니다.  
(앱 안 임베드 지도는 키 발급 후 4단계 이후 추가 가능)

---

## Android에서 실행 (권장)

1. Docker API 실행: `docker compose up -d`
2. PowerShell:

```powershell
cd C:\Users\psw00\OneDrive\Desktop\docker_exam2
.\mobile\scripts\run-android-dev.ps1
```

- `.env`의 `ATTENDANCE_APP_API_KEY`를 자동으로 `--dart-define`에 넣습니다.
- **에뮬레이터** 기본 API 주소: `http://10.0.2.2:3001/api`
- **실제 휴대폰**이면 스크립트 안 `$apiBase`를 PC Wi-Fi IP로 변경  
  예: `http://192.168.0.10:3001/api`
