# 로그인 (3번) — DB 회원

회원 아이디·비밀번호는 **PostgreSQL `members` 테이블**에 저장됩니다.  
로그인 성공 시 **세션 토큰**이 `member_sessions` 테이블에 기록됩니다.

## 임시 관리자 계정 (시드)

| 항목 | 값 |
|------|-----|
| **아이디** | `admin` |
| **비밀번호** | `green1234` |
| 표시 이름 | 관리자 |
| 권한 레벨 | 10 (관리자) |

> 실제 서비스 배포 전에는 반드시 비밀번호를 변경하세요.

## 사용 방법

1. `docker compose up -d --build`
2. 브라우저: http://localhost:8081/bbs/login.php
3. 위 아이디·비밀번호 입력 → **로그인**
4. 성공 시 이전 페이지(또는 홈)로 이동, 헤더에 **관리자님** · **로그아웃** 표시

## API

| 메서드 | URL | 설명 |
|--------|-----|------|
| POST | `/api/auth/login` | 로그인 (body: `mb_id`, `mb_password`, `auto_login`) |
| GET | `/api/auth/me` | 현재 회원 (Header: `Authorization: Bearer 토큰`) |
| POST | `/api/auth/logout` | 로그아웃 |

## DB에서 확인

```powershell
docker exec hanwhagreen-db psql -U hanwha -d hanwhagreen -c "SELECT mb_id, mb_name, mb_level FROM members;"
docker exec hanwhagreen-db psql -U hanwha -d hanwhagreen -c "SELECT mb_id, expires_at FROM member_sessions ORDER BY created_at DESC LIMIT 5;"
```

## 자동 로그인

- 체크 시: 토큰을 **localStorage**에 저장 (약 30일)
- 미체크 시: **sessionStorage** (브라우저 탭 닫으면 만료)

## 공지사항 관리 (관리자 전용)

로그인한 관리자(`level >= 10`)만 다음이 가능합니다.

- 공지 **글쓰기** · **수정** · **삭제**
- API: `POST` / `PUT` / `DELETE` `/api/notice` (Authorization: Bearer 토큰 필요)

일반 방문자는 공지 **목록·상세 조회**만 가능합니다.

### 온라인문의 비밀글

| 사용자 | 비밀글 열람 |
